# IDOCR — extraction notes (on-demand)

## Image request checklist (when user types IDOCR)

Ask for:
- A clear, in-focus photo/scan of the ID
- Preferably **front** (and **back** if the document has key fields on the back)
- No heavy glare; all 4 corners visible

## Common label hints

- Full name (local): look for labels like Name, 姓名, Nom, Nombre, ชื่อ-สกุล, Nama
- DOB: Date of birth, Birth, 出生日期, 生年月日, 出生, DOB
- Expiry: Date of expiry, Exp, 有効期限, 截止日期
- Issuing country: often shown as Country / Nationality / Issuing State; passports may show country code (e.g., HKG, CHN, JPN)

## Quality + verification

If the model/OCR output conflicts (e.g., two different DOBs), ask a follow-up question and mark conflicting fields as `[unclear]` until confirmed.
